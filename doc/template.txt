template.tsx работает почти так-же как layout, но

но template
- Создает новый экземпляр компонента при каждой навигации.
- не сохраняет состояние 
- Перезагружается, ререндерит компонент при каждом переходе.
- Используется для шаблонов, которые должны обновляться при переходе

layout
- Используется для элементов, которые не должны обновляться при переходах (например, общий хедер, футер, навигация).
